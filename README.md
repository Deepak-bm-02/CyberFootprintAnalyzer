# CyberFootprint Analyzer

CyberFootprint Analyzer is a Full Stack Cybersecurity web application built using Flask and MySQL.

## Features

- User Registration and Login
- Cyber Risk Analysis
- Risk Score Calculation
- Email Breach Detection
- Security Recommendations
- Risk History Tracking
- Risk Trend Graph using Chart.js

## Tech Stack

- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript
- Chart.js
